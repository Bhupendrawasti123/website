How This Website Works

This birthday celebration website shows random motivational messages with each visit or refresh.

Features:

  • Different inspirational birthday message on each load
  • Beautiful photo display with circular frame
  • Thematic icons matching each message
  • Continuous confetti celebration
  • "Another Wish" button to get new messages
  • Multiple color schemes to choose from
  • Special animations and effects

How to Personalize:

You can customize this page by adding parameters to the URL:

  • ?name=FirstName - Sets the birthday person's name
  • ?pic=ImageURL - Sets the photo to display
  • ?theme=theme-name - Sets initial theme (default, theme-sunset, theme-ocean, etc.)

Example: birthday.html?name=Sarah&pic=https://example.com/photo.jpg&theme=theme-ocean

If no parameters are provided, it will use default values.
